Module StaTz.Grid

type regular = private
| Regular

A 1d sampling grid is either regular (the most common case) or explicitly specified by an array of sampling times.

type explicit = private
| Explicit
type 'a t = private
| Regular : {
step : float;
start : int;
count : int;
} -> regular t
| Explicit : {
reals : float array;
} -> explicit t
type interval =
| Real_interval of {
start : float;
stop : float;
}
| Index_interval of {
start : int;
stop : int;
}

The intervals are understood to be inclusive.
