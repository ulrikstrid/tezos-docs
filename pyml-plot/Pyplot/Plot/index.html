module Axis : sig ... end

Plots are called "Axes" in matplotlib. For now we provide 2d & 3d scatterplots and histograms. We abstract monadically over matplotlib's statefulness.

type 'a t

The type of layouts of plots. This specifies how a plot should be displayed. matplotlib allows to compose plots together as subplots. We provide another monadic interface to abstract over matplotlib's statefulness.

We can't use the same monad as in the Axis module because the corresponding states are handled distinctly in matplotlib.
