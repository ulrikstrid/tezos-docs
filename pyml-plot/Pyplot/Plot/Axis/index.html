Module Plot.Axis

Plots are called "Axes" in matplotlib. For now we provide 2d & 3d scatterplots and histograms. We abstract monadically over matplotlib's statefulness.

val scatter_2d : xs:Matrix.t -> ys:Matrix.t -> color:color -> shape:shape -> (unit, dim2) t

2d scatter plot. xs stores the x-components of the points and ys the y-components of the points. xs and ys should be row vectors of the same dimensions. color specifies the color used for the plot. shape specifies the shape of each point in the scatter plot.
